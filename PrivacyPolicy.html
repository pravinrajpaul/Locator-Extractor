Privacy Policy

XPath & CSS Locator Extractor is a Chrome extension that runs entirely in your browser and does not collect, store, or transmit any personal data.

Data Usage

All operations (such as scanning webpages and extracting selectors) are performed locally. The extension does not send any data to external servers.

Permissions

  • activeTab / scripting: Used to scan the current tab for DOM elements on user request.
  • storage: Used only to store preferences locally in your browser.
  • downloads: Allows users to download extracted selectors to a local file, initiated manually.
  • host permissions: Required to access the webpage content, but only after user interaction.
